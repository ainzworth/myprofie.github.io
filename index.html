My Project

Overview

A random name generator for games products that creates exotic and unique names based on algorithms. Applying self-learning machine techniques or create a lexical rule for how names are generated

Motivation

More than 1000 games each year and thousands more of start-up companies is created are created. According to statistics, this percentage has received a rising in popularity in recent years. And many of these company need to have a name for their new branches which is catchy, short and fit well for their company’s purposes. Regarding the game products, this program can be used as a random name generator as well as a way to collect information from players so that the developers can understand more about their players in order to be adaptive and improve their game. Therefore, this project was born in order to alleviate the needs of society for a multi-purpose application.

Description

This application exists as a software that have to be paid in order to use

The user enters a name or a keyword that they want their name to be based on or relevant to it. The purpose is to get the name ideas so that the generator can create short, catchy names, the clearer the keywords, the better the results. There is also a filter system in order to make the name more suitable by categorizing it into many categories such as your favorite characters, film, etc. In order to understand the user need, this application also consists of a self-learning machine so the more data that it collects from the user, the better the results are.

This application can be used in a variety of way. It can exist in a form of a website to generate random name for free in order to popularize this application. Or it can be sold for companies that need this program for their own purposes.The application can also work as a system that collects data and information from users, store it and send it to the developers. This allow the creators to understand about their customers or game developers to know more about the players need and what changes should be made base on the data the system gathered.

Skills Required

Advance coding skills, require years of experience in coding. Knowing as many coding languages as possible such as python, C++,…

To create a self-learning machine you need to have: Knowledge about matrix, vector, statistics and calculus

Understand about basics concept of ML such as Linear algebra, probability theory, calculus, optimization theory, programming
